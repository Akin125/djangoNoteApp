# Django Note App

A simple Django application for creating and managing notes, containerized with Docker.

## Features

- Create, read notes
- User authentication
- SQLite database
- Docker containerization
- Admin panel to manage notes

## Tech Stack

- Django 6.0+
- Django REST Framework
- SQLite
- Docker & Docker Compose
- Python 3.12

## Prerequisites

- Docker
- Docker Compose
- Git
